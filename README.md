# ₹ OnYou

Split bills. Pay together.

## Overview

OnYou is a fintech-inspired web application that simplifies group expense management by allowing users to create shared bills, split expenses equally or custom, track payment status, and settle balances seamlessly.

## Features

- Create Bills
- Groups
- Quick Pay
- Settlements
- Invite Friends
- Merchant QR Scanning
- Payment Tracking

## Tech Stack

Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Node.js
- Express.js

Database:
- MongoDB
